# Code Examples

This directory contains runnable examples for each major OOP concept covered in the tutorial.

## Examples

1. **example_01_basic_class.py** - Basic class definition and object creation
2. **example_02_inheritance.py** - Inheritance and polymorphism
3. **example_03_magic_methods.py** - Magic methods for operator overloading

## How to Run

```bash
python3 example_01_basic_class.py
python3 example_02_inheritance.py
python3 example_03_magic_methods.py
```

## Learning Tips

- Read the code before running it
- Try modifying the examples
- Experiment with adding new methods or classes
- Break things and see what errors you get!
